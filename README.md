## Installation

### Local Setup

```bash
# Clone repository
git clone https://github.com/YOUR_USERNAME/metriq-ai.git
cd metriq-ai

# Create virtual environment
python -m venv venv

# Activate (Mac/Linux)
source venv/bin/activate

# Activate (Windows)
venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Create .env file
echo "OPENAI_API_KEY=your-key-here" > .env

# Run application
streamlit run app.py
